## Condition
This extension only accepts a local IP meaning the model must be accessible locally, however this shouldn't be an issue as this extension was developed using a `mini` model, meaning it can run with as little as 4GB of VRAM which most graphics cards have.

## Usage
After adding the extension via the method inbuilt into SillyTavern, access the extensions folder within your SillyTavern instance, there you will find a batch file, optionally you can copy it from the repository, this batch file SHOULD handle setting up ollama and the model.
